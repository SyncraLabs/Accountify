## 2. User Flows

### Onboarding
1.  **Sign Up**: User registers via Email/Password.
2.  **Profile Setup**: User adds a photo and bio.
3.  **First Habit**: User is prompted to create their first habit or use the AI Coach.

### The "Daily Loop"
1.  **Check-in**: User opens app to see daily tasks.
2.  **Execute**: User performs the habit in real life.
3.  **Track**: User toggles the habit on the dashboard.
4.  **Socialize**: User posts a "Done" update in the Squad chat.

### The "Falling Off" Flow
1.  **Inactivity**: User misses a day.
2.  **Notification**: App sends a "Streak Risk" alert.
3.  **Recovery**: User uses "Freeze Streak" or simply resumes to rebuild momentum.
